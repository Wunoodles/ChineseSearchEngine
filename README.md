##詞性分類
```
● 副詞 -> D
       -> DE(的)
● 動詞 -> V
       -> V_2(有)
● 名詞 -> N
       -> Nh(代名詞)
       -> Nf(量詞，包含Neqb、Neqa、Neu、Neu)
● 形容詞 -> A       
● 介系詞 -> P
● 連接詞 -> C
● 感嘆詞 -> I

```
